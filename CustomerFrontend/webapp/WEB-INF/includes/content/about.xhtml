About easyTravel

easyTravel provides a web portal which allows users to log in, search for journeys to various destinations, select special offers and promotional journeys and purchase a journey using a credit card.

The application demonstrates the functionality of Dynatrace APM products.

iOS App Installation

iOS 7.1+ requires the installation of the self-signed SSL-Certificate used by this site. To download it, click on this SSL-Certificate link and then follow instructions.
Additionally in iOS 10.3 you must manually trust this certificate: go to Settings -> General -> About -> Certificate Trust Settings and toggle: 'Enable Full Trust For Root Certificates'
After installing the application, the certificate can be deleted at Settings -> General -> Profiles -> Configuration Profiles.
Last step is to trust the application: go to Settings -> General -> Profiles & Device Managment, click on Dynatrace LLC enterprise APP and select Trust "Dynatrace LCC"
Certificate is generated automatically and it requires installed OpenSSL on Linux ( we provide OpenSSL binaries for Windows with easyTravel). Remember to set config.apacheWebServerSslHost property in easyTravelLocal.properties file to match your address.
